Comment héberger un site web facilement avec GitHub

Quelques étapes :

  • Créer un fichier html et css
  • créer un compte GitHub sur le site internet github.com
  • Sur GitHub : aller sur la petite croix en haut à droite pour créer un nouveau projet "new repository"
  • Compléter le nom du projet (par exemple : naomiacc.github.ic)
  • Cliquer sur "Create repository"
  • Dans l'en-tête : cliquer sur "uploading an existing file"
  • Choisir les fichiers à envoyer sur Github
  • Pour trouver le site web : copier/coller le nom du projet sur la barre de recherche d'un navigateur
  • Pour trouver l'adresse du site : aller dans "settings" section "GitHub Pages".

Remarque 1 : cette méthode d'hébergement ne vaut que pour les site statiques. Elle ne s'adresse pas aux site e-commerce ni aux blogs par exemple.

Remarque 2 : Le site HTML5UP donne des templates HTML5 responsives et gratuits, notammeent pour la création de portfolio.
